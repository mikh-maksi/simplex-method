На вході задача отримує матрицю коефіцієнтів обмежень **A_ij**, стовпчик правих частих обмежень **b_i** та рядок коефіцієнтів при цільовій функції **c_i**.  
На основі розмірності відповідних елементів додається:
* До **A_ij** дописується одинична матриця порядку, який дорівнює кількості рядків в вихідній матриці **A_ij**. Кількість рядків **m**, кількість стовпців **n**
* **Xbasis_i** - норме стовпців, що знаходяться в базиці **n+1**,**n+2**, ... , **n+m**
* **delta_i** - рякдок, що складається з 0 довжиною **n+m**
* **fi_i** - - рякдок, що складається з 0 довжиною **n+m**
* **Z_0** - дорівнює 0. 
* **c_i** - додається **m** нулів справа.

## Тест-кейс
1. Двоїста задача.
   1. Формулюємо базову постановку.
   2. Будуємо двоїсту задачу від базової.
   3. Передаємо дані для симплекс-методу (із додаванням всіх елементів).
   4. Отримаємо на виході симплекс-таблицю, яка склалася.
   5. Додаємо елемент обмеження, який робить таблицю невирішеною.
   6. Додаємо крок до симплекс-методу (додаємо необхідну кількість базисних стовпчиків).
   7. Вирішуємо симлпекс-метод.
